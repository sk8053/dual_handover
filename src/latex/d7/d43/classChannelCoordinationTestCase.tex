\subsection{Detailed Description}
This test case tests the channel coordination. In particular, it checks the following\+: 


\begin{DoxyItemize}
\item channel interval calculation including C\+CH Interval, S\+CH Interval, Guard Interval and Sync Interval
\item current interval state for current time and future time
\item channel coordination events notified at the correct time. 
\end{DoxyItemize}
