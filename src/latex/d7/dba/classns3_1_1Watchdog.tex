\subsection{Detailed Description}
A very simple watchdog operating in virtual time. 

The watchdog timer is started by calling Ping with a delay value. Once started the timer cannot be suspended, cancelled or shortened. It {\itshape can} be lengthened (delayed) by calling Ping again\+: if the new expire time (current simulation time plus the new delay) is greater than the old expire time the timer will be extended to the new expire time.

Typical usage would be to periodically Ping the \hyperlink{classns3_1_1Watchdog}{Watchdog}, extending it\textquotesingle{}s execution time. If the owning process fails to Ping before the \hyperlink{classns3_1_1Watchdog}{Watchdog} expires, the registered function will be invoked.

If you don\textquotesingle{}t ping the watchdog sufficiently often, it triggers its listening function.
