\subsection{Detailed Description}
The \hyperlink{classns3_1_1LteSimpleSpectrumPhy}{Lte\+Simple\+Spectrum\+Phy} models the physical layer of L\+TE This class is used to test Frequency Reuse Algorithms, it allow to get \hyperlink{classns3_1_1SpectrumValue}{Spectrum\+Value} from channel and pass it to test script by trace mechanism. When m\+\_\+cell\+Id is 0, all received signals will be traced, if m\+\_\+cell\+Id $>$ 0, only signals from specified Cell will be traced.
