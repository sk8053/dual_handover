\subsection{Detailed Description}
This class is very useful when user needs to collect statistics from P\+D\+CD and R\+LC. It automatically connects \hyperlink{classns3_1_1RadioBearerStatsCalculator}{Radio\+Bearer\+Stats\+Calculator} to appropriate trace sinks. Usually user do not use this class. All he/she needs to to do is to call\+: \hyperlink{classns3_1_1LteHelper_adc62a53efe0437cf09062c9d6a95af3d}{Lte\+Helper\+::\+Enable\+Pdcp\+Traces()} and/or \hyperlink{classns3_1_1LteHelper_abadfdd04d30b261e9b6f0846b4784928}{Lte\+Helper\+::\+Enable\+Rlc\+Traces()}.
