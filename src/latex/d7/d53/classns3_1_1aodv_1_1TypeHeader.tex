This method is used by \hyperlink{classns3_1_1Packet_a0961eccf975d75f902d40956c93ba63e}{Packet\+::\+Remove\+Header} to re-\/create a header from the byte buffer of a packet. The data read is expected to match bit-\/for-\/bit the representation of this header in real networks.

Note that data is not actually removed from the buffer to which the iterator points. Both \hyperlink{classns3_1_1Packet_a0961eccf975d75f902d40956c93ba63e}{Packet\+::\+Remove\+Header()} and \hyperlink{classns3_1_1Packet_aadc63487bea70945c418f4c3e9b81964}{Packet\+::\+Peek\+Header()} call \hyperlink{classns3_1_1aodv_1_1TypeHeader_a1008c3c92926976b0bcf863508f0073a}{Deserialize()}, but only the Remove\+Header() has additional statements to remove the header bytes from the underlying buffer and associated metadata.
