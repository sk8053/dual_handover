\subsection{Detailed Description}
Abstract \hyperlink{classns3_1_1Channel}{Channel} Base Class. 

A channel is a logical path over which information flows. The path can be as simple as a short piece of wire, or as complicated as space-\/time.

Subclasses must use \hyperlink{classns3_1_1Simulator_a86dbaef45a15a42365d7d2ae550449f6}{Simulator\+::\+Schedule\+With\+Context} to correctly update event contexts when scheduling an event from one node to another one.
