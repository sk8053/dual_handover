\subsection{Detailed Description}
\hyperlink{classns3_1_1Tag}{Tag} used to allow a fast identification of the packet. 

This tag is added by \hyperlink{classns3_1_1FlowMonitor}{Flow\+Monitor} when a packet is seen for the first time, and it is then used to classify the packet in the following hops.
