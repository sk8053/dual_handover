\subsection{Detailed Description}
Simple \hyperlink{classns3_1_1SpectrumPhy}{Spectrum\+Phy} implementation that sends customizable waveform. The generated waveforms have a given Spectrum Power Density and duration (set with the Set\+Resolution()) . The generator activates and deactivates periodically with a given period and with a duty cycle of 1/2.

This P\+HY model supports a single antenna model instance which is used for both transmission and reception (though received signals are discarded by this P\+HY).
