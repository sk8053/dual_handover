\subsection{Detailed Description}
Allocate positions from a deterministic list specified by the user. 

The first call to \hyperlink{classns3_1_1ListPositionAllocator_a39c1db898ea6db91c3bed8adf73f02b4}{List\+Position\+Allocator\+::\+Get\+Next} will return the first element of the list, the second call, the second element, and so on.
