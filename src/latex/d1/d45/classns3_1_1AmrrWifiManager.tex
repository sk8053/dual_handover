\subsection{Detailed Description}
A\+M\+RR Rate control algorithm

This class implements the A\+M\+RR rate control algorithm which was initially described in {\itshape I\+E\+EE 802.\+11 Rate Adaptation\+: A Practical Approach}, by M. Lacage, M.\+H. Manshaei, and T. Turletti. 

This R\+AA does not support HT, V\+HT nor HE modes and will error exit if the user tries to configure this R\+AA with a Wi-\/\+Fi M\+AC that has Vht\+Supported, Ht\+Supported or He\+Supported set.
