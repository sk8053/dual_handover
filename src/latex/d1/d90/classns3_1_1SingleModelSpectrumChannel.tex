Add a \hyperlink{classns3_1_1SpectrumPhy}{Spectrum\+Phy} to a channel, so it can receive packets. 

This method is used to attach a \hyperlink{classns3_1_1SpectrumPhy}{Spectrum\+Phy} instance to a \hyperlink{classns3_1_1SpectrumChannel}{Spectrum\+Channel} instance, so that the \hyperlink{classns3_1_1SpectrumPhy}{Spectrum\+Phy} can receive packets sent on that channel. Note that a \hyperlink{classns3_1_1SpectrumPhy}{Spectrum\+Phy} that only transmits (without receiveing ever) does not need to be added to the channel.

This method is to be implemented by all classes inheriting from \hyperlink{classns3_1_1SpectrumChannel}{Spectrum\+Channel}.
