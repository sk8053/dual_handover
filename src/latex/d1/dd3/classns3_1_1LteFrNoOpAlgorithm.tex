\subsection{Detailed Description}
FR algorithm implementation which simply does nothing. 

Selecting this FR algorithm is equivalent to disabling F\+FR. This is the default choice.

To enable FR, please select another FR algorithm, i.\+e., another child class of \hyperlink{classns3_1_1LteFfrAlgorithm}{Lte\+Ffr\+Algorithm}.
