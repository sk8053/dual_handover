\subsection{Detailed Description}
Simple \hyperlink{classns3_1_1SpectrumPhy}{Spectrum\+Phy} implementation that averages the spectrum power density of incoming transmissions to produce a spectrogram.

This P\+HY model supports a single antenna model instance which is used for reception (this P\+HY model never transmits).
