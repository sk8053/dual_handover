\subsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\\*
class ns3\+::\+Ptr$<$ T $>$}

Smart pointer class similar to {\ttfamily boost\+::intrusive\+\_\+ptr}. 

This smart-\/pointer class assumes that the underlying type provides a pair of {\ttfamily Ref} and {\ttfamily Unref} methods which are expected to increment and decrement the internal reference count of the object instance. You can add {\ttfamily Ref} and {\ttfamily Unref} to a class simply by inheriting from \hyperlink{classns3_1_1SimpleRefCount}{ns3\+::\+Simple\+Ref\+Count}.

This implementation allows you to manipulate the smart pointer as if it was a normal pointer\+: you can compare it with zero, compare it against other pointers, assign zero to it, etc.

It is possible to extract the raw pointer from this smart pointer with the \hyperlink{classns3_1_1Ptr_a27a6a8208868e66947c81f77f5bd751b}{Get\+Pointer()} and \hyperlink{classns3_1_1Ptr_a9914221293005018082da554c6059d5e}{Peek\+Pointer()} methods.

If you want to store a {\ttfamily new} object into a smart pointer, we recommend you to use the \hyperlink{group__ptr_ga0406dd2d929ceac05ce39d4579fa3e4b}{Create()} template functions to create the object and store it in a smart pointer to avoid memory leaks. These functions are really small convenience functions and their goal is just is save you a small bit of typing.
