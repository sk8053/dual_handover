\subsection{Detailed Description}
Class to represent a 3D function expression plot. 

Since the function expression is not escaped, styles and extras could just as well be included in the expression string. The only difference to \hyperlink{classns3_1_1Gnuplot2dFunction}{Gnuplot2d\+Function} is the splot command string.
