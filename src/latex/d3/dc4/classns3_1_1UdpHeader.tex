\subsection{Detailed Description}
\hyperlink{classns3_1_1Packet}{Packet} header for U\+DP packets. 

This class has fields corresponding to those in a network U\+DP header (port numbers, payload size, checksum) as well as methods for serialization to and deserialization from a byte buffer.

Force the U\+DP checksum to a given value. 

This might be useful for test purposes or to restore the U\+DP checksum when the U\+DP header has been compressed (e.\+g., in 6\+Lo\+W\+P\+AN). Note that, normally, the header checksum is calculated on the fly when the packet is serialized.

When this option is used, the U\+DP checksum is written in the header, regardless of the global Checksum\+Enabled option.

\begin{DoxyNote}{Note}
The checksum value must be a big endian number.
\end{DoxyNote}
