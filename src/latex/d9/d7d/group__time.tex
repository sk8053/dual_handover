\subsection{Detailed Description}
Management of virtual time in real world units. 

The underlying simulator is unit agnostic, just dealing with dimensionless virtual time. Models usually need to handle time in real world units, such as seconds, and conversions/scaling between different units, between minutes and seconds, for example.

The convenience constructors in the \hyperlink{group__timecivil}{Standard Units} module make it easy to create Times in specific units.

The \hyperlink{classns3_1_1Time_ac89165ba7715b66017a49c718f4aef09}{Time\+::\+Set\+Resolution()} function allows a one-\/time change of the base resolution, before \hyperlink{classns3_1_1Simulator_a84be982e6d03b62c3dc3303c75a9b909}{Simulator\+::\+Run()}.
