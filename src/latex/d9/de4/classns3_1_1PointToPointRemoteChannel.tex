\subsection{Detailed Description}
A Remote Point-\/\+To-\/\+Point \hyperlink{classns3_1_1Channel}{Channel}. 

This object connects two point-\/to-\/point net devices where at least one is not local to this simulator object. It simply override the transmit method and uses an M\+PI Send operation instead.
