\subsection{Detailed Description}
The Extended Supported Rates Information Element

This class knows how to serialise and deserialise the Extended Supported Rates Element that holds (non-\/\+HT) rates beyond the 8 that the original Supported Rates element can carry. 

The {\ttfamily \hyperlink{classns3_1_1SupportedRates}{Supported\+Rates}} class still records all the rates, and an instance of {\ttfamily \hyperlink{classns3_1_1ExtendedSupportedRatesIE}{Extended\+Supported\+Rates\+IE}} lies within {\ttfamily \hyperlink{classns3_1_1SupportedRates}{Supported\+Rates}}.
