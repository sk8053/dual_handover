\subsection{Detailed Description}
These are the internal implementation functions for the Attribute system.

Module code shouldn\textquotesingle{}t need to call these directly. Instead, see \hyperlink{group__attributehelper}{Attribute Helper}.

There are three versions of Do\+Make\+Accessor\+Helper\+One\+:
\begin{DoxyItemize}
\item With a member variable\+: \hyperlink{group__attributeimpl_gaee5b20c72f968868a1bef8f01db84263}{Do\+Make\+Accessor\+Helper\+One(\+U T\+::$\ast$)}
\item With a class get functor\+: Do\+Make\+Accessor\+Helper\+One(U(\+T\+::$\ast$)(void) const)
\item With a class set method\+: Do\+Make\+Accessor\+Helper\+One(void(\+T\+::$\ast$)(U))
\end{DoxyItemize}

There are two pairs of Do\+Make\+Accessor\+Helper\+Two (four total)\+:
\begin{DoxyItemize}
\item Taking two arguments, a set method and get functor, in either order,
\item With set methods returning {\ttfamily void} or {\ttfamily bool}. 
\end{DoxyItemize}
