This test verifies the accuracy of the Geographic\+To\+Cartesian\+Coordinates() method in the Geographic\+Positions class, which converts earth geographic/geodetic coordinates to E\+C\+EF Cartesian coordinates. To do so, it compares the values generated from the method to values generated from the M\+A\+T\+L\+AB function geodetic2ecef(), which is part of the M\+A\+T\+L\+AB Mapping Toolbox, using the built-\/in earth reference\+Sphere, G\+R\+S80 reference\+Ellipsoid, and W\+G\+S84 reference\+Ellipsoid in M\+A\+T\+L\+AB. A description of the M\+A\+T\+L\+AB function can be found at this webpage\+: \href{http://www.mathworks.com/help/map/ref/geodetic2ecef.html}{\tt http\+://www.\+mathworks.\+com/help/map/ref/geodetic2ecef.\+html} Values are compared using 216 test cases for each of the three earth spheroid models.
