Generates uniformly distributed random points (in E\+C\+EF Cartesian coordinates) within a given altitude above earth\textquotesingle{}s surface centered around a given origin point (on earth\textquotesingle{}s surface, in geographic/geodetic coordinates) within a given distance radius (using arc length of earth\textquotesingle{}s surface, not pythagorean distance). Distance radius is measured as if all generated points are on earth\textquotesingle{}s surface (with altitude = 0). Assumes earth is a perfect sphere.
