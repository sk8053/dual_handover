\subsection{Detailed Description}
Implement the data structure representing a Traffic\+Flow\+Template \hyperlink{classns3_1_1Packet}{Packet} Filter. See 3\+G\+PP TS 24.\+008 version 8.\+7.\+0 Release 8, Table 10.\+5.\+162/3\+G\+PP TS 24.\+008\+: Traffic flow template information element

With respect to the \hyperlink{classns3_1_1Packet}{Packet} Filter specification in the above doc, the following features are N\+OT supported\+:
\begin{DoxyItemize}
\item I\+Pv6 filtering (including flow labels)
\item I\+P\+Sec filtering
\item filter precedence field is not evaluated, hence it is recommended to setup the T\+F\+Ts within a P\+DP context such that T\+F\+Ts are mutually exclusive 
\end{DoxyItemize}
