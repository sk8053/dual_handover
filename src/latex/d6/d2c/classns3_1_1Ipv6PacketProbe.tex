\subsection{Detailed Description}
This class is designed to probe an underlying \hyperlink{namespacens3}{ns3} Trace\+Source exporting a packet, an I\+Pv6 object, and an interface. This probe exports a trace source \char`\"{}\+Output\char`\"{} with arguments of type Ptr$<$const Packet$>$, Ptr$<$\+Ipv6$>$, and uint32\+\_\+t. The Output trace source emits a value when either the trace source emits a new value, or when Set\+Value () is called.
