\subsection{Detailed Description}
read and write tag data 

This class allows subclasses of the \hyperlink{classns3_1_1Tag}{ns3\+::\+Tag} base class to serialize and deserialize their data through a stream-\/like A\+PI. This class keeps track of the \char`\"{}current\char`\"{} point in the buffer and advances that \char`\"{}current\char`\"{} point everytime data is written. The in-\/memory format of the data written by this class is unspecified.

If the user attempts to write more data in the buffer than he allocated with \hyperlink{classns3_1_1Tag_a01c8efdea943fd8cce2ef5d4d67eefa6}{Tag\+::\+Get\+Serialized\+Size}, he will trigger an N\+S\+\_\+\+A\+S\+S\+E\+RT error.
