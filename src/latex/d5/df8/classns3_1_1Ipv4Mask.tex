\subsection{Detailed Description}
a class to represent an \hyperlink{classns3_1_1Ipv4}{Ipv4} address mask 

The constructor takes arguments according to a few formats. \hyperlink{classns3_1_1Ipv4Mask}{Ipv4\+Mask} (\char`\"{}255.\+255.\+255.\+255\char`\"{}), \hyperlink{classns3_1_1Ipv4Mask}{Ipv4\+Mask} (\char`\"{}/32\char`\"{}), and \hyperlink{classns3_1_1Ipv4Mask}{Ipv4\+Mask} (0xffffffff) are all equivalent.
