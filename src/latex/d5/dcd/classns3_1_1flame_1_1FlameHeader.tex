Note that data is not actually removed from the buffer to which the iterator points. Both \hyperlink{classns3_1_1Packet_a0961eccf975d75f902d40956c93ba63e}{Packet\+::\+Remove\+Header()} and \hyperlink{classns3_1_1Packet_aadc63487bea70945c418f4c3e9b81964}{Packet\+::\+Peek\+Header()} call \hyperlink{classns3_1_1flame_1_1FlameHeader_ada911cbb255eb829e842812dce4b2fc7}{Deserialize()}, but only the Remove\+Header() has additional statements to remove the header bytes from the underlying buffer and associated metadata.
