\subsection{Detailed Description}
Pure virtual base class for the interface between ns-\/3 and the parallel communication layer being used. 

Each type of parallel communication layer is required to implement this interface. This interface is called through the \hyperlink{classns3_1_1MpiInterface}{Mpi\+Interface}.
