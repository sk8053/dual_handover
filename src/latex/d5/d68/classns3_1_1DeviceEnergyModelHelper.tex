\subsection{Detailed Description}
Creates \hyperlink{classns3_1_1DeviceEnergyModel}{Device\+Energy\+Model} objects. 

This class helps to create and install \hyperlink{classns3_1_1DeviceEnergyModel}{Device\+Energy\+Model} onto \hyperlink{classns3_1_1NetDevice}{Net\+Device}. A \hyperlink{classns3_1_1DeviceEnergyModel}{Device\+Energy\+Model} is connected to a \hyperlink{classns3_1_1NetDevice}{Net\+Device} (or P\+HY object) by callbacks. Note that \hyperlink{classns3_1_1DeviceEnergyModel}{Device\+Energy\+Model} objects are {\itshape not} aggregated onto the node. They can be accessed through the \hyperlink{classns3_1_1EnergySource}{Energy\+Source} object, which {\itshape is} aggregated onto the node.
