\subsection{Detailed Description}
Testing a handover algorithm, verifying that it selects the right target cell when more than one options available. 

Part of the {\ttfamily lte-\/handover-\/target} test suite.

The test case will run a 1-\/second L\+T\+E-\/\+E\+PC simulation using the parameters provided to the constructor function.
