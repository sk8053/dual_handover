\subsection{Detailed Description}
Abstract class to store a plot line to be used by \hyperlink{classns3_1_1Gnuplot}{ns3\+::\+Gnuplot}. 

This class contains a reference counted data object in m\+\_\+data. The data object contains different structs derived from struct \hyperlink{structns3_1_1GnuplotDataset_1_1Data}{Data} by subclasses.
