\subsection{Detailed Description}
interface for antenna radiation pattern models 

This class provides an interface for the definition of antenna radiation pattern models. This interface is based on the use of spherical coordinates, in particular of the azimuth and inclination angles. This choice is the one proposed \char`\"{}\+Antenna Theory -\/ Analysis
and Design\char`\"{}, C.\+A. Balanis, Wiley, 2nd Ed., see in particular section 2.\+2 \char`\"{}\+Radiation pattern\char`\"{}.
