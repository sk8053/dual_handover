\subsection{Detailed Description}
Antenna model based on a parabolic approximation of the main lobe radiation pattern. 

This class implements the parabolic model as described in some 3\+G\+PP document, e.\+g., R4-\/092042

A similar model appears in

George Calcev and Matt Dillon, \char`\"{}\+Antenna Tilt Control in C\+D\+M\+A Networks\char`\"{} in Proc. of the 2nd Annual International Wireless Internet Conference (W\+I\+C\+ON), 2006

though the latter addresses also the elevation plane, which the present model doesn\textquotesingle{}t.
