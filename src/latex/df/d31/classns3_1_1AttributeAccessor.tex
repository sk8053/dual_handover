\subsection{Detailed Description}
allow setting and getting the value of an attribute. 

The goal of this class is to hide from the user how an attribute is actually set or get to or from a class instance. Implementations of this base class are usually provided through the Make\+Accessor\+Helper template functions, hidden behind an A\+T\+T\+R\+I\+B\+U\+T\+E\+\_\+\+H\+E\+L\+P\+E\+R\+\_\+$\ast$ macro.
