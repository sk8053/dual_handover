\subsection{Detailed Description}
Threshold for event evaluation. 

For R\+S\+R\+P-\/based threshold, the actual value is (value -\/ 140) d\+Bm. While for R\+S\+R\+Q-\/based threshold, the actual value is (value -\/ 40) / 2 dB. This is in accordance with section 9.\+1.\+4 and 9.\+1.\+7 of 3\+G\+PP TS 36.\+133.
