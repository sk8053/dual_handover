\subsection{Detailed Description}
A U\+DP server, receives U\+DP packets from a remote host. 

U\+DP packets carry a 32bits sequence number followed by a 64bits time stamp in their payloads. The application uses the sequence number to determine if a packet is lost, and the time stamp to compute the delay.
