\subsection{Detailed Description}
A\+L\+O\+HA M\+AC Protocol, the simplest M\+AC protocol for wireless networks.

Packets enqueued are immediately transmitted. This M\+AC attaches a \hyperlink{classns3_1_1UanHeaderCommon}{Uan\+Header\+Common} to outgoing packets for address information. (The type field is not used)
