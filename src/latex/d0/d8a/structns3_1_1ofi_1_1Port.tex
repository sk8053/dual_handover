\subsection{Detailed Description}
\hyperlink{structns3_1_1ofi_1_1Port}{Port} and its metadata. 

We need to store port metadata, because Open\+Flow dictates that there exists a type of request where the \hyperlink{classns3_1_1ofi_1_1Controller}{Controller} asks for data about a port, or multiple ports. Otherwise, we\textquotesingle{}d refer to it via Ptr$<$\+Net\+Device$>$ everywhere.
