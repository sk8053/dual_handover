\subsection{Detailed Description}
an Inet address class 

This class is similar to inet\+\_\+sockaddr in the B\+SD socket A\+PI. i.\+e., this class holds an \hyperlink{classns3_1_1Ipv4Address}{Ipv4\+Address} and a port number to form an ipv4 transport endpoint.
