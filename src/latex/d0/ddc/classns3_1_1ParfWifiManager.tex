\subsection{Detailed Description}
P\+A\+RF Rate control algorithm

This class implements the P\+A\+RF algorithm as described in {\itshape Self-\/management in chaotic wireless deployments}, by Akella, A.; Judd, G.; Seshan, S. and Steenkiste, P. in Wireless Networks, Kluwer Academic Publishers, 2007, 13, 737-\/755 \href{http://www.cs.odu.edu/~nadeem/classes/cs795-WNS-S13/papers/enter-006.pdf}{\tt http\+://www.\+cs.\+odu.\+edu/$\sim$nadeem/classes/cs795-\/\+W\+N\+S-\/\+S13/papers/enter-\/006.\+pdf}

This R\+AA does not support HT, V\+HT nor HE modes and will error exit if the user tries to configure this R\+AA with a Wi-\/\+Fi M\+AC that has Vht\+Supported, Ht\+Supported or He\+Supported set.
