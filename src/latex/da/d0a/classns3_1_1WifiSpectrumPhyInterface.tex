\subsection{Detailed Description}
This class is an adaptor between class \hyperlink{classns3_1_1SpectrumWifiPhy}{Spectrum\+Wifi\+Phy} (which inherits from \hyperlink{classns3_1_1WifiPhy}{Wifi\+Phy}) and class \hyperlink{classns3_1_1SpectrumChannel}{Spectrum\+Channel} (which expects objects derived from class \hyperlink{classns3_1_1SpectrumPhy}{Spectrum\+Phy} to be connected to it).

The adaptor is used only in the receive direction; in the transmit direction, the class \hyperlink{classns3_1_1SpectrumWifiPhy}{Spectrum\+Wifi\+Phy} constructs signal parameters and directly accesses the \hyperlink{classns3_1_1SpectrumChannel}{Spectrum\+Channel}
