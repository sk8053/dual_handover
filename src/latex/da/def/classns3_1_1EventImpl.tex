\subsection{Detailed Description}
A simulation event. 

Each subclass of this base class represents a simulation event. The \hyperlink{classns3_1_1EventImpl_a5ce561412ea89951be67fc19d389df32}{Invoke()} method will be called by the simulation engine when it reaches the time associated to this event. Most subclasses are usually created by one of the many \hyperlink{classns3_1_1Simulator_a671882c894a08af4a5e91181bf1eec13}{Simulator\+::\+Schedule} methods.
