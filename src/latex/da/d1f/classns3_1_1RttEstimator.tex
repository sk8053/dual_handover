\subsection{Detailed Description}
Base class for all R\+TT Estimators. 

The R\+TT Estimator class computes an estimate of the round trip time observed in a series of \hyperlink{classns3_1_1Time}{Time} measurements. The estimate is provided in the form of an estimate and a sample variation. Subclasses can implement different algorithms to provide values for the estimate and variation.
