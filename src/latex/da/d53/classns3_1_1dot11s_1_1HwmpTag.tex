\subsection{Detailed Description}
Hwmp tag implements interaction between H\+W\+MP protocol and Mesh\+Wifi\+Mac. 

Hwmp tag keeps the following\+:
\begin{DoxyEnumerate}
\item When packet is passed from Hwmp to 11s\+M\+AC\+:
\begin{DoxyItemize}
\item retransmitter address,
\item T\+TL value,
\end{DoxyItemize}
\item When packet is passed to Hwmp from 11s\+M\+AC\+:
\begin{DoxyItemize}
\item lasthop address,
\item T\+TL value,
\item metric value (metric of link is recalculated at each packet, but routing table stores metric obtained during path discovery procedure) 
\end{DoxyItemize}
\end{DoxyEnumerate}
