\subsection{Detailed Description}
Set of frequency values implementing the domain of the functions in the Function Space defined by \hyperlink{classns3_1_1SpectrumValue}{Spectrum\+Value}. Frequency values are in Hz. It is intended that frequency values are non-\/negative, though this is not enforced.
