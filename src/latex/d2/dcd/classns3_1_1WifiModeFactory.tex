\subsection{Detailed Description}
create \hyperlink{classns3_1_1WifiMode}{Wifi\+Mode} class instances and keep track of them. 

This factory ensures that each \hyperlink{classns3_1_1WifiMode}{Wifi\+Mode} created has a unique name and assigns to each of them a unique integer.
