\subsection{Detailed Description}
\hyperlink{classns3_1_1Ipv6QueueDiscItem}{Ipv6\+Queue\+Disc\+Item} is a subclass of \hyperlink{classns3_1_1QueueDiscItem}{Queue\+Disc\+Item} which stores I\+Pv6 packets. \hyperlink{classns3_1_1Header}{Header} and payload are kept separate to allow the queue disc to manipulate the header, which is added to the packet when the packet is dequeued.
