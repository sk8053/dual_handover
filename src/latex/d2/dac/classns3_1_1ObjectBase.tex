\subsection{Detailed Description}
Anchor the ns-\/3 type and attribute system. 

Every class which wants to integrate in the ns-\/3 type and attribute system should derive from this base class. This base class provides\+:
\begin{DoxyItemize}
\item A way to associate an \hyperlink{classns3_1_1TypeId}{ns3\+::\+Type\+Id} to each object instance.
\item A way to set and get the attributes registered in the \hyperlink{classns3_1_1TypeId}{ns3\+::\+Type\+Id}. 
\end{DoxyItemize}
