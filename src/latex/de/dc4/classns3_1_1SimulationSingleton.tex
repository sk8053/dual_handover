\subsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\\*
class ns3\+::\+Simulation\+Singleton$<$ T $>$}

This singleton class template ensures that the type for which we want a singleton has a lifetime bounded by the simulation run lifetime. That it, the underlying type will be automatically deleted upon a call to \hyperlink{classns3_1_1Simulator_a2a056e59a6623225df0957eda0ee8252}{Simulator\+::\+Destroy}.

For a singleton with a lifetime bounded by the process, not the simulation run, see \hyperlink{classns3_1_1Singleton}{Singleton}.
