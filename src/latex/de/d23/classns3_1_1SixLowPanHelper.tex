Install the Six\+Lo\+W\+P\+AN stack on top of an existing \hyperlink{classns3_1_1NetDevice}{Net\+Device}. 

This function requires a set of properly configured Net\+Devices passed in as the parameter \char`\"{}c\char`\"{}. The new Net\+Devices will have to be used instead of the original ones. In this way these Six\+Lo\+W\+P\+AN devices will behave as shims between the Net\+Devices passed in and I\+Pv6.

Note that only I\+Pv6 (and related protocols, such as I\+C\+M\+Pv6) can be transmitted over a 6\+Lo\+W\+P\+AN interface. Any other protocol (e.\+g., I\+Pv4) will be discarded by 6\+Lo\+W\+P\+AN.

Other protocols (e.\+g., I\+Pv4) could be used on the original Net\+Devices with some limitations. See the manual for a complete discussion.

\begin{DoxyNote}{Note}
I\+Pv6 stack must be installed {\itshape after} Six\+Lo\+W\+P\+AN, using the Six\+Lo\+W\+P\+AN Net\+Devices. See the example in the examples directory.
\end{DoxyNote}
