\subsection{Detailed Description}
This system test program creates different test cases with a single e\+NB and several U\+Es, all having the same Radio Bearer specification. In each test case, the U\+Es see the same S\+I\+NR from the e\+NB; different test cases are implemented obtained by using different S\+I\+NR values and different numbers of U\+Es. The test consists on checking that the obtained throughput performance is equal among users is consistent with the definition of token bank fair queue scheduling
