\subsection{Detailed Description}
Protocol trailer serialization and deserialization. 

Every Protocol trailer which needs to be inserted or removed from a \hyperlink{classns3_1_1Packet}{Packet} instance must derive from this base class and implement the pure virtual methods defined here.
