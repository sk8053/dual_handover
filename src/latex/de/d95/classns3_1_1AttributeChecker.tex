\subsection{Detailed Description}
Represent the type of an attribute. 

Each type of attribute has an associated unique \hyperlink{classns3_1_1AttributeChecker}{Attribute\+Checker} subclass. The type of the subclass can be safely used by users to infer the type of the associated attribute. i.\+e., we expect binding authors to use the checker associated to an attribute to detect the type of the associated attribute.

Most subclasses of this base class are implemented by the A\+T\+T\+R\+I\+B\+U\+T\+E\+\_\+\+H\+E\+L\+P\+E\+R\+\_\+\+H\+E\+A\+D\+ER and A\+T\+T\+R\+I\+B\+U\+T\+E\+\_\+\+H\+E\+L\+P\+E\+R\+\_\+\+C\+PP macros.
