\subsection{Detailed Description}
The packet header for the \hyperlink{classns3_1_1Packet}{Packet} Data Convergence Protocol (P\+D\+CP) packets. 

This class has fields corresponding to those in an P\+D\+CP header as well as methods for serialization to and deserialization from a byte buffer. It follows 3\+G\+PP TS 36.\+323 \hyperlink{classns3_1_1Packet}{Packet} Data Convergence Protocol (P\+D\+CP) specification.
