\subsection{Detailed Description}
Byte tag using by Anim to uniquely identify packets. 

When Anim receives a Tx Notification we tag the packet with a unique global uint64\+\_\+t identifier before recording Tx information When Anim receives Rx notifications the tag is used to retrieve Tx information recorded earlier
