\subsection{Detailed Description}
implement the C\+A\+RA rate control algorithm

Implement the C\+A\+RA algorithm from\+: J. Kim, S. Kim, S. Choi, and D. Qiao. \char`\"{}\+C\+A\+R\+A\+: Collision-\/\+Aware Rate Adaptation for I\+E\+E\+E 802.\+11 W\+L\+A\+Ns.\char`\"{} 

Originally implemented by Federico Maguolo for a very early prototype version of ns-\/3.

This R\+AA does not support HT, V\+HT nor HE modes and will error exit if the user tries to configure this R\+AA with a Wi-\/\+Fi M\+AC that has Vht\+Supported, Ht\+Supported or He\+Supported set.
