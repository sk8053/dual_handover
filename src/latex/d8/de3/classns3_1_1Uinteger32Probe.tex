\subsection{Detailed Description}
This class is designed to probe an underlying \hyperlink{namespacens3}{ns3} Trace\+Source exporting an uint32\+\_\+t. This probe exports a trace source \char`\"{}\+Output\char`\"{} of type uint32\+\_\+t. The Output trace source emits a value when either the trace source emits a new value, or when Set\+Value () is called.

The current value of the probe can be polled with the Get\+Value () method.
