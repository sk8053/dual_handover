\subsection{Detailed Description}
This class implements the Access Stratum (AS) Service Access Point (S\+AP), i.\+e., the interface between the \hyperlink{classns3_1_1EpcUeNas}{Epc\+Ue\+Nas} and the \hyperlink{classns3_1_1LteUeRrc}{Lte\+Ue\+Rrc}. In particular, this class implements the Provider part of the S\+AP, i.\+e., the methods exported by the \hyperlink{classns3_1_1LteUeRrc}{Lte\+Ue\+Rrc} and called by the \hyperlink{classns3_1_1EpcUeNas}{Epc\+Ue\+Nas}.
