\subsection{Detailed Description}
Helper class used to remember already seen packets and detect duplicates. 

Currently duplicate detection is based on uinique packet ID given by \hyperlink{classns3_1_1Packet_a1f212c825b50e54d94f5b9ae99592e6a}{Packet\+::\+Get\+Uid} () This approach is known to be weak and should be changed.
