\subsection{Detailed Description}
helps to create \hyperlink{classns3_1_1WaveNetDevice}{Wave\+Net\+Device} objects 

This class can help to create a large set of similar \hyperlink{classns3_1_1WaveNetDevice}{Wave\+Net\+Device} objects and to configure a large set of their attributes during creation.

Generally \hyperlink{classns3_1_1WaveHelper}{Wave\+Helper} with default configuration will create devices with 7 M\+AC entities,1 P\+HY entity and a multiple-\/channel scheduler for to deal with these entities.

If users can make sure on which channel this W\+A\+VE device will work, they can set specific channel numbers to save resources of unused channels as below\+: \hyperlink{classns3_1_1WaveHelper}{Wave\+Helper} helper = \hyperlink{classns3_1_1WaveHelper_aa3b90ecb4657cdeaf77e67532dd2e690}{Wave\+Helper\+::\+Default} (); uint32\+\_\+t channels\mbox{[}\mbox{]} = \{C\+CH, S\+C\+H1\}; std\+::vector$<$uint32\+\_\+t$>$ channels\+Vector (channels, channels + 2); helper.\+helper.\+Create\+Mac\+For\+Channel (channels\+Vector);

If users can create other channel scheduler such as \char`\"{}\+Another\+Scheduler\char`\"{} which can assign channel access in the context of more P\+HY entities, they can use this helper to create W\+A\+VE devices as below\+: \hyperlink{classns3_1_1WaveHelper}{Wave\+Helper} helper = \hyperlink{classns3_1_1WaveHelper_aa3b90ecb4657cdeaf77e67532dd2e690}{Wave\+Helper\+::\+Default} (); helper.\+helper.\+Create\+Mac\+For\+Channel (\hyperlink{classns3_1_1ChannelManager_a4b4ce965b76235f8dc9fba47357908c3}{Channel\+Manager\+::\+Get\+Wave\+Channels} ()); helper.\+Create\+Phys (2); // or other number which should be less than 7 helper.\+Set\+Channel\+Scheduler (\char`\"{}ns3\+::\+Another\+Scheduler\char`\"{}); helper.\+Set\+Remote\+Station\+Manager (\char`\"{}ns3\+::\+Constant\+Rate\+Wifi\+Manager\char`\"{}); // or other rate control algorithms
